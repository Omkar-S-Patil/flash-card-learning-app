# Flash Card Learning App

A Python-based flash card application built using Tkinter and Pandas to help users learn French vocabulary interactively.

## Features
- Random French word generation
- Automatic card flipping after 3 seconds
- Tracks learned words
- Progress saving using CSV files
- Interactive GUI

## Technologies Used
- Python
- Tkinter
- Pandas

## Project Structure
- main.py → Main application logic
- data/ → Vocabulary datasets
- images/ → GUI assets

## How to Run
1. Install dependencies:
   pip install pandas

2. Run:
   python main.py
